# cps-meetup-30

This is the monorepo created at the [Campinas Meetup #30](https://www.meetup.com/pt-BR/campinas-frontend/events/267776112/) by me (@_luistak)

## Usage

Installation

```sh
yarn
```

Building and serving the app

```sh
yarn start
```
